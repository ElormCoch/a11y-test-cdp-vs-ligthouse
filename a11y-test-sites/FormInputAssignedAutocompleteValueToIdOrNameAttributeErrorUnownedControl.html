FormInputAssignedAutocompleteValueToIdOrNameAttributeErrorUnownedControl

Issue Type: Page Error

Lighthouse Detection: ❌ Not Detected

Test Example: Chromium Source

Description: Form input elements that are not owned by a form element (unowned controls) incorrectly use autocomplete keyword values as their id or name attributes. This misuse affects browser autofill behavior and can prevent proper form processing for standalone form controls.

Test Case

Issue: Text inputs without any form of labeling
