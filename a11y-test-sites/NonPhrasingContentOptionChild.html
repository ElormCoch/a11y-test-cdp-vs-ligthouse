NonPhrasingContentOptionChild

Issue Type: Page Error

Lighthouse Detection:❌ Not Detected

DevTools CDP Audits Detection ✅ Detected

Test Example: Chromium Source

Description: Non-phrasing content elements (like block-level elements: div, p, h1-h6) are incorrectly nested within <option> elements. Option elements should only contain phrasing content (inline text and simple formatting) to maintain proper dropdown structure and accessibility.

Test Case

Issue: The option element allows only non-interactive phrasing content, text, and div elements as its children. The semantics of non-phrasing content elements do not make sense as children of an option, and such semantics will largely be ignored by assistive technology since they are inappropriate in this context. Consider removing or changing such elements to one of the allowed phrasing content elements.
